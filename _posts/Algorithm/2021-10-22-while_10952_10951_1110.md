```python
#백준_10952
while True:
	A,B=map(int,input().split())
	if (A==0 and B==0):
		break
	else:
		print(A+B)
```

```python
#백준_10951
while True:
    try:
        A, B=map(int,input().split())
        print(A+B)
    except:
        break
```

```python
#백준_1110
N=int(input())
check=N
k=0
while 1: 
  a=N//10
  b=N%10
  New=b*10+(a+b)%10
  k+=1
  N=New
  if check==New:
    break
print(k)

########다른 풀이
num = int(input())
check = num
new_num = 0
temp = 0
count = 0
while True:
    temp = num//10 + num%10
    new_num = (num%10)*10 + temp%10
    count += 1
    num = new_num
    if new_num == check:
        break
print(count)
```



어떤 정해진 횟수 동안 반복하는 것은 for문

계속 반복하면서 어떤 조건이 되면 멈추는 것을 구현할 때는 while문이 좋을 것 같다.

while문은 아직 낯설다.
